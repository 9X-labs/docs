The **Display** sub-tab is the cosmetic + locale layer. None of these settings change *what* you see — only *how* it's presented.

## Theme

| Option | Effect |
| --- | --- |
| **Dark** | Black backgrounds, orange accents. The Znox default. |
| **Light** | White backgrounds, same orange accents. |
| **Auto** | Follows your OS preference. |

## Density

How tightly information is packed.

| Option | Use when |
| --- | --- |
| **Compact** | Big monitor, want max info on screen |
| **Comfortable** | Default — good readability/density balance |
| **Spacious** | Smaller screen or accessibility preference |

## Language

UI language. Currently:

- **English** (default)
- **Arabic** (RTL — layouts flip automatically)

The Telegram bot supports the same set of languages.

## Timezone

How timestamps are displayed across the app. Defaults to your browser's timezone. Override if you want UTC or a specific region.

## Currency

Which fiat currency is shown alongside crypto values for context. USD by default. Doesn't change trading — only the display label on USD-equivalent figures.

## Expert mode

A master toggle that exposes advanced controls across the app, including:

- Custom gas limits in [Terminal Settings](/web-app/terminal/settings).
- Manual route override in order entry.
- Front-running protection toggle on individual trades.

<Warning>
  Expert mode is for users who know what each control does. The defaults are calibrated for a reason — turn this on only if you have specific needs the defaults don't meet.
</Warning>

## Animations

Toggle UI animations (price ticks, page transitions, hover effects). Off can help on low-power devices or for accessibility.

## Save Changes

Like other settings sub-tabs, hit **Save Changes** at the bottom. Most display changes apply immediately on save.
