**Top Traders** ranks the most successful traders on Solana. The data is global — it includes wallets that don't necessarily use Znox, sourced from on-chain analytics.

## Columns

| Column | Meaning |
| --- | --- |
| Rank | Position in the leaderboard |
| Trader | Username or address (truncated) |
| P&L | Profit/loss in USD over the timeframe |
| Win rate | % of profitable trades |
| Trades | Number of executed trades |
| Volume | Total volume in USD |

## What "global" means

This is not just Znox users — it includes any Solana wallet that's been profitable in the chosen timeframe. The leaderboard is a discovery tool for finding wallets worth tracking.

## Pivot to action

From any row you can:

- **Track this wallet** — adds the trader to your [Track Wallets](/web-app/traders-hub/track-wallets) view so you see their trades live.
- **Open profile** — view their public profile if they're a Znox user.
- **Copy address** — copy the wallet for use elsewhere.
