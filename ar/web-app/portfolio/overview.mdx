تبويب **Portfolio** هو موطن أموالك. يدرج كلّ محفظة في حسابك، يتتبّع P&L لكلّ محفظة، ويمنحك أربع عمليّات أساسيّة على الأموال: إيداع، سحب، توزيع، تجميع.

لمعرفة لماذا يستخدم Znox نموذج محافظ متعدّدة، راجع [المحفظة المتعدّدة](/ar/concepts/multi-wallet).

## التخطيط

الصفحة بلوحَين:

- **اليسار** — قائمة كلّ محافظك بإحصاءات لكلٍّ.
- **اليمين** — عرض تفصيليّ للمحفظة المختارة، بتبويبات [Holdings / History / Orders](/ar/web-app/portfolio/holdings-history-orders).

## أعمدة قائمة المحافظ

| العمود | المعنى |
| --- | --- |
| Wallet | الاسم اللطيف + المفتاح العامّ المختصر (مع زرّ نسخ) |
| Invested | إجمالي القيمة المُستثمَرة بالعملة الأساسيّة |
| PnL | الربح/الخسارة المحقَّق + غير المحقَّق |
| W/L | عدد الفائزات / الخاسرات (المراكز المغلقة فقط) |
| SOL | رصيد SOL الحاليّ |

## بطاقة الإحصاءات الجانبيّة

للمحفظة المختارة:

- **الرصيد الإجماليّ** — مبلغ SOL + المعادل بالدولار.
- **إجمالي PnL USD** بقيمته المطلقة والنسبيّة.
- **الأرباح غير المحقَّقة** — المراكز المفتوحة فقط.
- **إجمالي الحجم** — حجم التداول مدى الحياة.

## إضافة محافظ

زرّ **Add Wallet** (أعلى القائمة) يُنشئ محفظة Solana جديدة في حسابك. تبدأ المحفظة فارغةً — مَوِّلها عبر Deposit أو Distribute.

## تقويم PnL

زرّ **PNL Calendar** يفتح [عرض حراريًّا](/ar/web-app/portfolio/pnl-calendar) لتاريخ P&L اليوميّ.

## الخصوصيّة

أيقونة العين أعلى اللوحة الجانبيّة تبدّل رؤية الرصيد — مفيدة عند مشاركة شاشتك.

## تخطيط الجوّال

على الجوّال، قائمة المحافظ قابلة للطيّ. اضغط محفظة لتوسيعها إلى عرض التفاصيل؛ اضغط سهم الرجوع للعودة إلى القائمة.
