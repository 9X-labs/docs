عند اختيار محفظة في [Portfolio](/ar/web-app/portfolio/overview)، تُظهر اللوحة اليمنى ثلاثة تبويبات.

## Holdings

المراكز المفتوحة لهذه المحفظة — رموز اشتريتها ولم تبعها بالكامل.

| العمود | المعنى |
| --- | --- |
| الرمز | الاسم + الرمز + وقت آخر نشاط |
| Bought / MC | إجمالي ما استثمرت + القيمة السوقيّة عند الدخول |
| Sold / MC | الخروجات المحقَّقة + القيمة السوقيّة عند الخروج |
| Bal | رصيد الرمز الحاليّ + المعادل بالدولار |
| UPnL | P&L غير محقَّق ($ و %) |
| Total Profit | المحقَّق + غير المحقَّق مجتمعَيْن |
| Holding Duration | الوقت منذ أوّل شراء |

### المفلترات

- **إخفاء المراكز المغلقة** — حذف الرموز التي بعتها بالكامل.
- **إخفاء السيولة المنخفضة** — حذف الرموز التي مجمَّعاتها رفيعة جدًّا للخروج بثقة.
- **إخفاء "لم تشترِ"** — حذف الـ airdrops أو الرموز التي وصلت بدون شراء مقابل.
- **عرض القديمة** — تعليم المراكز الأقدم من عتبة.
- **عرض المخفيّة (N)** — استعادة الرموز التي أخفيتها سابقًا عبر قائمة السياق.

### بيع سريع

كلّ صفّ به أزرار نسبة جاهزة — `25 / 50 / 75 / 100%` — لخروجات جزئيّة سريعة. الزرّ الأيمن للنقر للحصول على قائمة سياق (بيع، نقل، إخفاء).

## History

كلّ شراء / بيع / تبادل على هذه المحفظة، مرتَّبًا زمنيًّا.

| العمود | المعنى |
| --- | --- |
| Side | BUY / SELL / SWAP |
| الرمز | ما الذي تداولته |
| الكميّة | الرمز + العملة الأساسيّة |
| السعر | سعر التنفيذ |
| الطابع الزمنيّ | متى أُغلقت الصفقة |
| الحالة | مؤكَّدة / فاشلة |
| TX | رابط مستكشف Solana |

زرّ تصفية يحدّد النطاق إلى **الكلّ / الشراءات / المبيعات**. التصدير إلى CSV من قائمة الإجراءات.

## Orders

**أوامر Limit** المفتوحة لهذه المحفظة (فقط إن وضعت أيًّا منها — راجع [تنفيذ الأوامر](/ar/concepts/order-execution)). كلّ صفّ يُظهر السعر الهدف، الجهة، الكميّة، وزرّ الإلغاء.

إن لم تضع أمر Limit مطلقًا، هذا التبويب فارغ.
