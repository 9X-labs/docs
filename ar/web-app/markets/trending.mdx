**Trending** هو واجهة فحص الرموز الرئيسيّة. اختر إطارًا زمنيًّا، تصفّح الجدول، اضغط على أيّ شيء مثير للاهتمام.

## الإطارات الزمنيّة

الأزرار في الأعلى — `5m / 30m / 1h / 12h / 24h` — تغيّر الفترة التي يُحسب عليها "الرائج". النوافذ الأقصر تُظهر العملات الميمية المتقلّبة؛ النوافذ الأطول تُظهر الرموز ذات الزخم المستدام.

## الأعمدة

الكثير محشور في الجدول. هذا ما يعنيه كلّ عنوان:

| العمود | المعنى |
| --- | --- |
| `#` | الترتيب في الإطار الحاليّ |
| الرمز | الاسم + الرمز + الشعار |
| السعر | السعر الحاليّ |
| **CHG** | تغيّر السعر في الإطار |
| **MCAP** | القيمة السوقيّة |
| **LIQ** | سيولة المجمَّع بالدولار |
| **VOL** | حجم التداول في الإطار |
| **HOLD** | عدد الحاملين |
| **B/S** | عدد الشراءات / المبيعات في الإطار (مثلًا `9/12` = 9 شراءات، 12 بيعًا) |
| **LP%** | % من المعروض في مجمَّع السيولة. الأعلى = float أصحّ |
| **TOP10** | % من المعروض لدى أكبر 10 محافظ. القيم العالية تشير إلى مخاطر الحيتان |
| **DEV** | % لدى ناشر العقد. القيم العالية تشير إلى مخاطر التفريغ |
| **SNP** | عدد الـ snipers — محافظ اشترت في الكتلة الأولى |
| **SOCIALS** | شارات Twitter / website إن وُجدت |
| **VERIFY** | ✓ إن كان العقد معتمَدًا / يجتاز فحوص الأمان |

## أزرار جانب اليسار

كلّ صفّ به زرّان:

- **JUP** — يفتح عرض Jupiter Aggregator لهذا الرمز في تبويب جديد.
- **CA** — ينسخ عنوان العقد.

## كيف تقرأ الجدول بسرعة

1. **CHG** + **VOL** يُظهران الزخم. حركة كبيرة بحجم منخفض = شريط رفيع، مشكوك.
2. **TOP10** + **DEV** خطر تركيز. >50% للأعلى-10 يعني عددًا قليلًا من الحيتان يتحكّمون في حركة السعر.
3. **SNP** يقفز على الرموز التي تجمّع عليها الـ snipers عند الإطلاق — أحيانًا إشارة alpha، أحيانًا ضخّ منسَّق.
4. **LP%** أقلّ من 50% يعني أنّ معظم المعروض ليس في المجمَّع. مجمَّعات LP الأعمق تتداول بـ slippage أقلّ.

<Tip>
  حين تكون نسبة `B/S` تميل إلى البيع لكنّ `CHG` إيجابيّ، فالشراءات القليلة كبيرة جدًّا — ربّما الناشر أو حوت يمتصّ المعروض. انظر إلى **TOP10** للتحقّق.
</Tip>
