The Znox Telegram bot ([@znox_trading_betabot](https://t.me/znox_trading_betabot)) is a parallel surface to the web app. Anything you can do on the web — trade, manage wallets, broadcast, earn — you can do via Telegram messages and inline buttons.

## When to use the bot vs the web

<CardGroup cols={2}>
  <Card title="Use the bot when" icon="bolt">
    You're on mobile, in a group chat, or want to act without switching apps. Quick Ape from a group broadcast is the killer use case.
  </Card>
  <Card title="Use the web when" icon="browser">
    You need charts, the security audit panel, multi-wallet operations, the PnL calendar, or the deep token analytics in Markets.
  </Card>
</CardGroup>

## Account & wallet sharing

One Znox account = one set of wallets, accessible from both surfaces. Sign up on the web, open the bot — your wallets are already there. Or sign up via the bot, then log in on the web with the same Telegram account.

Settings (slippage, MEV, Jito tier, presets) sync between both surfaces in real-time.

## Languages

The bot is bilingual — **English** and **Arabic**. The active language is configurable in `/settings`. RTL is fully supported in Arabic mode.
